Security Measures

Technical and organisational controls in place as of April 2026.

Encryption in Transit

TLS 1.2+ enforced on all endpoints via Nginx and Traefik. HTTP redirects to HTTPS automatically. Certificates auto-renewed via Let's Encrypt.

Access Control

Per-tenant Virtual Key isolation in LiteLLM. Per-agent scoped keys via OpenFGA relationship model. No cross-tenant data access possible.

Authorization

OpenFGA for relationship-based access (who owns what). OPA for conditional policy evaluation (budget, DPDP, expiry, rate limits) on every request.

Error Tracking

GlitchTip self-hosted for error monitoring. Stack traces captured per agent. No user PII in error payloads. Registration disabled — invite only.

Non-Root Containers

All sidecar images (playwright, classifier, translator) run as non-root users. OCI image annotations on all custom builds.

Secrets Management

No secrets committed to git. All credentials in server .env. Rotation policy: immediately on exposure. Agent keys shown once at creation only.

Audit Logging

Every LLM call traced in Langfuse with agent_id, tenant_id, model, token counts. OpenFGA logs all authorization decisions. GlitchTip captures policy denials.

Private Node Option

Dedicated VPS deployment. Customer is Data Fiduciary. All data stays within customer infrastructure. DPA available on request.
